# IMDB SQL Database Analysis

## Overview
This project uses PostgreSQL and SQL to analyze large-scale IMDB datasets.

## Objectives
- Query and analyze movie datasets
- Perform joins and aggregations
- Optimize SQL queries
- Analyze relational database structures

## Technologies Used
- PostgreSQL
- SQL
- Jupyter Notebook

## Skills Demonstrated
- SQL Queries
- Joins
- Window Functions
- Query Optimization
- Database Analysis
